Animal Crossing Clock An open-source clock that plays Animal Crossing music, by Sam Carson.

About

This is a small open-source web applet I made that plays music from Animal Crossing, based upon real-time hours. You should check out the original, Crossing Time, which this took heavy inspiration from. This was made for the sake of learning new programming skills.
